

# Problem Statement
This project is used to sort manual laptop under 40,000 rupees which is very much time-consuming and irritating,this project require users to browse multiple pages and copy-paste details repeatedly.

# Scope
- This Scrapes Flipkart laptop listings under 40,000 rupees
- This Extracts product names, prices, and descriptions
- This Processes first two pages of results
- Exports to CSV format

# Target Users
- This project mainly targets Students/Parents to sort a budget laptop
- This is mainly for Price-conscious buyers
- Anyone needing quick laptop comparisons

# Key Features
- Fetches faster the data collection (2 pages)
- Simple CSV output
- Basic error handling
- No external dependencies beyond standard libraries
